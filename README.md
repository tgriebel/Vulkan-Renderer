# Vulkan-Renderer

Work-in-progress.

Features:
* Depth-prepass
* Shadow maps
* PBR Opaque
* Transparents
* Post-process
* Multiple shaders
* Stencil effects
* Asset and scene management
* Model instancing
* Runtime Object manipulation
* Shader hotswap

Partial Support:
* Raytracer and software rasterizer
* MSAA
* Env-maps
